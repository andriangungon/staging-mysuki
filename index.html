Paano Gamitin?

Madaling matutunan kung paano gamitin ang My Suki!

i-download sa Playstore or Appstore ang “My Suki.”

Gumawa ng account gamit ang iyong mobile number para matanggap ang inyong One-Time password.

Click “add suki ” icon at i-add ang vendor code ng accredited supermarket.

Piliin ang mga produktong gusto mong bilhin.

Pindutin ang deliver button, ilagay ang address ng inyong tindahan, at antayin ang confirmation mula sa amin.

Hintayin ang delivery nito sa inyong sari-sari store!
